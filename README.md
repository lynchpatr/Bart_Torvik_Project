# Bart_Torvik_Project
- Analyzing how different Bart Torvik data filters and metric weightings affect NCAA tournament bracket predictions. I generated 40 brackets using conference, non‑conference, and all‑games stats, varied metric emphasis, and multiple tie‑breaking buffers of weighted scores to compare bracket score outcomes.

## Project Overview
  **Data Filters:** Conference-only, non-conference, all-games
- Half of bracket is filtered incon or noncon stats based on matchups;alrigjt other half is unfiltered
  
**Metric Weightings:** Offense, defense, shooting %, rebounding, turnovers
- # Tie breaker stats:
    - **Offense:** Offensive efficiency
    - **Defense:** Defensive efficiency
    - **Shooting Percentage:** Effective field goal percentage
    - **Rebounding:** Defensive rebounding percentage  
    - **Turnovers:** Turnover rate

  **Tie-Breaking Buffers:** 0, 1, 3, 5
- i.e If weighted score of Team A is within 1 of Team B in the offense skewed bracket, offensive efficiency breaks the tie. Each bracket type will have brackets with a buffer of 0, 1, 3, 5 in the weighted score.
  
  **Goal:** Compare bracket scoring performance across analytical setups to get a high bracket score
